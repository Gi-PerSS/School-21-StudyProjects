Подробное описание

Главный класс приложения, управляющий интерфейсом и логикой работы с 3D-моделями.

Этот класс отвечает за загрузку моделей, выполнение трансформаций (перемещение, поворот, масштабирование), настройку отображения, создание анимаций и сохранение скриншотов или GIF-анимаций.
