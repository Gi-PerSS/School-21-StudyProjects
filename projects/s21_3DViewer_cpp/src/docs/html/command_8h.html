Подробное описание

Заголовочный файл, содержащий объявления классов команд для выполнения трансформаций над моделью.

Этот файл содержит объявления абстрактного класса Command и его потомков, которые реализуют паттерн Command. Каждый класс-команда инкапсулирует конкретное действие (например, перемещение, поворот или масштабирование модели).
