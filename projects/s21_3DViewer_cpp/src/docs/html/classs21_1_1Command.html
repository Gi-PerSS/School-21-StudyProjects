Подробное описание

Абстрактный базовый класс для всех команд.

Класс Command определяет интерфейс для выполнения команд над моделью. Все конкретные команды наследуются от него.

pure virtual

Выполняет команду над моделью.

Аргументы
modelСсылка на объект ParserModel, над которым выполняется команда.

Замещается в s21::RotateXCommand, s21::RotateYCommand, s21::RotateZCommand, s21::MoveXCommand, s21::MoveYCommand, s21::MoveZCommand и s21::ScaleCommand.
