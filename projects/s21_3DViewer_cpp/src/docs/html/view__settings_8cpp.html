Подробное описание

Реализация класса Settings для управления настройками отображения.

Этот файл содержит реализацию методов класса Settings, который представляет диалоговое окно для настройки параметров отображения 3D-модели. Класс позволяет сохранять и загружать настройки, а также применять их к OpenGL-виджету.
