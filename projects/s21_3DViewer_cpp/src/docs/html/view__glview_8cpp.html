Подробное описание

Реализация класса GLView для отрисовки 3D-моделей с использованием OpenGL.

Этот файл содержит реализацию методов класса GLView, который отвечает за отрисовку 3D-моделей, обработку событий мыши и колеса, а также применение настроек отображения.
