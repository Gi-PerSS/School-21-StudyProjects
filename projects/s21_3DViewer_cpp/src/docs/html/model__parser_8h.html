Подробное описание

Заголовочный файл, содержащий объявления классов для работы с 3D-моделями.

Этот файл содержит объявления структуры ObjData и класса ParserModel, которые используются для загрузки, хранения и модификации данных 3D-моделей. Класс ParserModel также реализует паттерн Observer для уведомления наблюдателей об изменениях данных модели.
