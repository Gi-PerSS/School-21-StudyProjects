Подробное описание

Реализация методов класса ParserModel для работы с 3D-моделями.

Этот файл содержит реализацию методов класса ParserModel, который отвечает за загрузку, хранение и модификацию данных 3D-модели. Класс также реализует паттерн Observer для уведомления наблюдателей об изменениях данных.
